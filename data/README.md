# Data folder

This repository does not contain individual-level or private data.

The analysis uses secondary public indicators entered directly in the R Markdown workflow. Generated analysis datasets are created when the R Markdown file is rendered.

Primary public sources to cite in the manuscript and repository record:

1. IARC GLOBOCAN 2022 cancer indicators for the Republic of the Congo.
2. ICO/IARC HPV Information Centre country report for the Republic of the Congo.
3. WHO cervical cancer country profile for the Republic of the Congo.
4. WHO cervical cancer elimination targets: 90% vaccination, 70% screening, and 90% treatment.
5. World Bank or UN public population and health-system indicators where used.

Do not add private files, author emails, signed forms, reviewer comments, grant documents, or patient records to this folder.
